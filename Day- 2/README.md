# Day 2 – Input, Output & Operators in Java

# Topics Covered
    • Taking input from users
    • Printing output in Java
    • Java Operators

# Inputs in Java-
    Inputs can be taken using-

    1) Scanner class
        • Belongs to java.util package.
        • Used to take input from the user.

    2) BufferedReader class
        • Belongs to java.io package.
        • Used to read input efficiently using character streams.

# Outputs in Java-
    Output can be displayed using:

    1) print()
        • Prints output without moving to a new line.

    2) println()
        • Prints output and moves the cursor to the next line.

    3) printf()
        • Used to print formatted output.
