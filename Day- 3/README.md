# Conditional Statements
    Conditional statements are used to execute different blocks of code based on specific conditions.

1. If Statement-
    • Executes a block of code when a condition is true.

2. If-Else Statement-
    • Executes one block when the condition is true and another block when the condition is false.

3. Else-If Ladder-
    • Used to check multiple conditions sequentially.

4. Nested If Statement-
    • An if statement inside another if statement.

5. Switch Statement-
    • Used to execute code based on multiple possible values.

6. Enhanced Switch-
    • Modern switch expression introduced in newer Java versions with improved syntax.

7. Ternary Operator-
    • Short-hand alternative to simple if-else statements.
    • Syntax- condition ? expression1 : expression2;
