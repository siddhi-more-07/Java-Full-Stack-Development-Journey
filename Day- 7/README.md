### String Methods

    1) length() - Returns string length 
    2) charAt() - Returns character at a specific index
    3) toUpperCase() - Converts string to uppercase
    4) toLowerCase() - Converts string to lowercase
    5) trim() - Removes leading and trailing spaces
    6) equals() - Compares string contents
    7) equalsIgnoreCase() - Compares strings ignoring case
    8) compareTo() - Lexicographical comparison
    9) concat() - Joins two strings
    10) contains() - Checks if a substring exists
    11) startsWith() - Checks starting characters
    12) endsWith() - Checks ending characters
    13) substring() - Extracts a portion of a string
    14) replace() - Replaces characters or substrings
    15) replaceAll() - Replaces using regular expressions
    16) indexOf() - Finds first occurrence
    17) lastIndexOf() - Finds last occurrence 
    18) isEmpty() - Checks whether the string is empty
    19) split() - Splits a string into an array
